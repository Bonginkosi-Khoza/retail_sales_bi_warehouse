# Retail Sales Data Warehouse (SQL)
A professional SQL data warehouse project using real retail data.  
Includes staging, dimensional modeling, ETL scripting, data quality fixing, and analysis queries.

## Overview
This project implements a mini enterprise-style data warehouse using SQL and SQLite.
Multiple raw data sources are integrated into a star schema to support analytical querying
and business intelligence reporting.

## Business Problem
Retail sales data is distributed across multiple systems, limiting integrated analysis.
This project centralizes sales, customer, and product data into an analytics-optimized model.

## Architecture
- Staging tables for raw ingestion
- Star schema with fact and dimension tables
- SQL-based ETL transformations
- Incremental data loading
